# Transcoder
A web interface to transcode video files with Handbrake

## Dependencies
#### Handbrake CLI
`sudo apt install handbrake-cli`
#### Crystal
`curl -sSL https://dist.crystal-lang.org/apt/setup.sh | sudo bash`
`sudo apt install crystal`

## Build
#### Debug
`./build.sh`
#### Release
`./build.sh -r`

## Run
`./transcoder`
